Alias assigned to the Borehole by other authorities.<br/>Often special 'reference' identifier and name are assigned to a Borehole by the authority that maintains the reference dataset to ensure uniqueness.<br/>But the same Borehole can be identified in other information system(s) using alternative identifier and name.  This can be amplified when state level agencies merge/split or change data management technologies. <br/>Alias still refer to the same Borehole.<br/>
