The referent provides the link between the 3D world and the 1D view (the linear referencing system). Depending on the BoreholeReferencingMethod, it may be the “zero” of the positioning within the BoreholeLRS.<br/>The referent can either be a well-known feature (e.g. BoreholeCollar) described with the corresponding semantics or a coordinate (point geometry).<br/>Note : this class is an adaptation of LR_Referent<br/>
