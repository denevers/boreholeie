Describes the 'Linear Referencing System' that frames all the locations of the BoreholeEvent(s) composing BoreholeEventCollections. This LRS consists of the linear element being measured, the method of measurement and the referent relative to which all locations are measured.<br/>Note : this class is an adaptation of ISO 19148 LR_PositionExpression (see ISO 19148 Figure 14).<br/>
